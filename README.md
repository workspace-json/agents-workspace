# workspace-json/agents-workspace

Minimal repository for the `agents-workspace` reference. This repo contains a small sample `workspace.json` and a README providing links to the canonical spec and the documentation site.

This repository is intentionally minimal: the full specification lives in https://github.com/workspace-json/spec and the rendered documentation site is at https://github.com/workspace-json/website (deployed at https://workspacejson.dev).

Sample files included:

- `workspace.json` — a minimal example of a v0.1 `workspace.json` file
- `.agents/workspace.json` — same sample in the canonical location

If you'd like additional examples or CI checks here, tell me what you want included.
